## üöÄ Quick Start

### Option 1: Use the Website (Recommended)

1. Visit [companyportalicons.com](https://companyportalicons.com)
2. Search or browse for your app
3. Click "Preview" to see how it looks
4. Click "Download" to get the PNG file
5. Upload to your Intune app configuration

### Option 2: Clone the Repository

```bash
git clone https://github.com/MrtnRQL/Companyportaliconsdotcom.git
cd Companyportaliconsdotcom
```

All icons are located in:
- `assets/icons/aaron-parker/` - Icons from Aaron Parker's repository
- `assets/icons/` - Official source icons

## üí° Usage

### In Microsoft Intune

1. **Navigate to your app** in Intune admin center
2. **Edit app properties**
3. **Upload icon** - Select the downloaded PNG file
4. **Save** - Icon will appear in Company Portal immediately

### Recommended Icon Specifications

- **Format**: PNG (with transparency)
- **Dimensions**: 512x512 pixels (recommended)
- **File Size**: < 1MB
- **Background**: Transparent

All icons in this repository meet these specifications!
